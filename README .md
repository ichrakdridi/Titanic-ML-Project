# Titanic Machine Learning Project

## Description
Ce projet consiste à analyser et prédire la survie des passagers du Titanic. 
Le projet suit les étapes suivantes : 

- Nettoyage des données et gestion des valeurs manquantes (Week 2)
- Analyse exploratoire des données (EDA) (Week 1)
- Feature engineering (création de nouvelles features)
- Préparation des données pour la modélisation Machine Learning

## Dataset
- Source : `titanic (1).csv` .
- Contenu : informations sur les passagers (classe, âge, sexe, tarif, nombre de frères/sœurs/conjoints, nombre de parents/enfants, etc.) et la variable cible `survived`.

## Objectifs
- Comprendre la structure et la qualité des données
- Visualiser les distributions et corrélations entre variables
- Imputer les valeurs manquantes et encoder les variables catégorielles
- Créer de nouvelles features pour améliorer la prédictio
